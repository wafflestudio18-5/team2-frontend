# Team2 Frontend

Team2 프론트엔드입니다. Medium을 클론코딩한 [**Wadium**](https://www.wadium.shop/)입니다.

### 페이지 목록

- `/main` 메인 페이지
- `/edit` 글 작성 페이지
- `/story/:storyid` 글 보는 페이지
- `/search` 검색 페이지
- `/me/stories` 자신의 글 목록 페이지
- `/me/settings` 자신의 정보 수정 페이지
- `/user/:userid` 유저 글 목록 페이지
- `/user/:userid/about` 유저 정보 페이지
